# Claude Agent Guide

이 리포지토리에서 Claude(claude.ai/code, Claude CLI)는 아래 순서로 문서를 읽고 작업해야 합니다.

1. `AGENTS.md`  
   - 전체 문서 구조와 역할, 진입 순서를 설명합니다.
2. `AGENT_RULES.md`  
   - 멀티 에이전트 공통 규칙과 코드/문서 편집 규칙을 정의합니다.
3. `PROJECT_OVERVIEW.md`  
   - 프로젝트 개요, 아키텍처, 폴더 구조, 핵심 타입/모듈, 빌드/테스트 정보를 설명합니다.
4. `TODO.md`  
   - 현재 진행 중인 작업 목록과 상태를 관리합니다.

규칙/정책/구조/TODO를 변경할 때는,  
반드시 위 문서들 중 **해당되는 파일을 직접 수정**해야 합니다.  

이 파일(`claude.md`)에는 이 안내 외의 프로젝트 규칙을 두지 않습니다.
